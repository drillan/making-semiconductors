# 概要

このページは、スライド本編（`slides/`）の導入部・全体構成に対応する詳細ページ。
バリューチェーン全体の見取り図と、業界の分業構造、そして本編を貫く「日本の強み」という
通底テーマを、スライドより詳しく解説する。

## このドキュメントの読み方

`docs/` は、スライド本編を補足するための通常の Sphinx（Furo テーマ）サイト。各ページは
スライドの各部に対応する。

| ページ | 対応するスライドの部 |
| --- | --- |
| `overview`（本ページ） | 導入（第0部） |
| `design` | 設計（第1部） |
| `frontend` | 前工程（第2部） |
| `backend` | 後工程（第3部） |
| `industry` | 業界俯瞰（第4部） |
| `sources` | データ出典・確度区分の一覧 |
| `glossary` | 用語集 |

スライドは「1スライド1メッセージ」を原則に要点だけを示しており、各スライド末尾の出典行は
簡潔な要約にとどめている。数値の根拠や確度区分の詳細、企業の位置づけの背景説明はこちらの
詳細ページと `sources` ページに集約している。スライドを読んで気になった項目があれば、
対応するページの該当セクションを参照してほしい。

数値・シェアの扱いについては、本編・本ドキュメントとも共通の方針を取る。

```{note}
確度の高い数値（複数の一次情報・調査が一致するもの）は断定的に記述する。一方、民間調査の
推計や出典間で幅がある「概況」の数値は「約」を付けて丸め、断定を避ける。実装時点で出典が
確定していない一部の数値（AMATのイオン注入シェア、KLAの計測・検査指標、住友ベークライトに
よる京セラEMC事業買収の詳細、Heraeusのボンディングワイヤ全体シェアの%）は、本編・本ドキュメント
のいずれでも数値を明記していない。詳細は `sources` ページを参照。
```

## 半導体の種類

本編は先端ロジックとメモリを主軸に据える。パワー半導体は種類紹介と業界俯瞰で別系統の
サプライチェーンとして触れるにとどめ、製造フローの詳細には立ち入らない。

| 種類 | 用途・特徴 | 代表企業 |
| --- | --- | --- |
| ロジック | 演算処理を担う。微細化競争の主戦場 | TSMC / NVIDIA / Intel / AMD |
| DRAM | 主記憶。高速だが電源を切るとデータが消える揮発性メモリ | Samsung / SK hynix / Micron |
| NAND | 大容量ストレージ向けの不揮発性メモリ。積層段数を競う「層数競争」が焦点 | Samsung / Kioxia / SK hynix / Micron |
| パワー | 電力変換・制御を担う。自動車・産業機器向けが中心 | Infineon / STMicro / ローム |

## バリューチェーン全体像

半導体は、回路を「設計」し、シリコンウェハーに回路パターンを作り込む「前工程（ウェハー
プロセス）」を経て、個々のチップに切り出し外部と接続・保護する「後工程（パッケージング）」
に至る。これらすべての工程を、装置と材料を供給する企業群が下支えする構造になっている。

```{mermaid}
:caption: 半導体バリューチェーンの全体像
:align: center

flowchart LR
  subgraph SUP["装置・材料（下支え）"]
    EQ["装置企業"]
    MT["素材企業"]
  end
  D["設計"] --> F["前工程"] --> B["後工程"] --> M["市場（ファウンドリ・メモリ・OSAT等）"]
  EQ -.-> F
  EQ -.-> B
  MT -.-> F
  MT -.-> B
```

各部の内容は以下の通り。

- **設計（第1部）**: 仕様策定から論理設計・回路設計・レイアウト・検証までの流れ。EDA
  ツール・IP・命令セット（ISA）が土台を成す。→ `design` ページ
- **前工程（第2部）**: ウェハーに成膜・露光・エッチング・イオン注入・平坦化・洗浄・計測を
  繰り返し、回路パターンを何層にも積み重ねる。装置・材料の主要企業が最も多く登場する部。
  → `frontend` ページ
- **後工程（第3部）**: ウェハーを個々のチップに切り出し、パッケージに封止し、テストする。
  先端パッケージング（チップレット・HBM 等）の重要性が近年急速に高まっている。
  → `backend` ページ
- **業界俯瞰（第4部）**: 装置・素材・ファウンドリ・メモリ・OSAT・パワー半導体を横断的に
  俯瞰し、地政学的な論点にも触れる。→ `industry` ページ

## 分業構造：IDM／ファブレス／ファウンドリ／OSAT

半導体産業には、設計から製造までを自社で垂直統合する企業（**IDM**: Integrated Device
Manufacturer）と、工程ごとに専業化した企業群による**水平分業型**の2つの事業モデルが
共存している。

```{mermaid}
:caption: IDMと水平分業型の対比
:align: center

flowchart LR
  subgraph IDM["IDM（垂直統合型）"]
    I1["設計"] --> I2["製造（前工程・後工程）"]
  end
  subgraph SPLIT["水平分業型"]
    Fa["ファブレス：設計に特化"] -->|製造委託| Fo["ファウンドリ：製造専業"]
    Fo --> O["OSAT：組立・検査専業"]
  end
```

| 事業モデル | 特徴 | 代表企業 |
| --- | --- | --- |
| IDM | 設計から製造まで自社で一貫して行う。設備投資負担が大きい | Intel、Samsung（メモリ・ロジック両方）、ローム |
| ファブレス | 設計に特化し、製造はファウンドリに委託する | NVIDIA / Qualcomm / AMD / Apple / Broadcom |
| ファウンドリ | 他社設計チップの製造受託に専業化する | TSMC / Samsung（ファウンドリ事業） / Intel（近年参入） / Rapidus |
| OSAT | ダイシング・ボンディング・封止・テストなど後工程を受託する | ASE / Amkor / JCET |

1980年代までは IDM が主流だったが、設計と製造それぞれの専業化が進んだことで、現在は
水平分業型がロジック半導体の主流になっている。特に先端ロジックでは、製造装置・材料への
投資額が桁違いに大きくなったため、量産規模で装置・材料コストを回収できるファウンドリに
製造が集約される傾向が強い。分業構造の詳しい力学は `design` ページで解説する。

## 「日本の強み」という通底テーマ

本編は各工程を追う中で、日本企業が世界シェアで大きな存在感を持つ分野を随所で取り上げる。
これは「日本礼賛」が目的ではなく、**業界構造を理解する軸**として使う。日本企業は完成品の
最終製造（先端ロジックのファウンドリ）では存在感が薄い一方、**素材・特定の専業装置・
マスク周辺**では世界的な「隠れたチャンピオン」が多く存在するという非対称な構造が、
業界俯瞰の理解を助けるためだ。

確度高で断定できる代表例（詳細は各部ページ・`industry` ページ・`sources` ページを参照）:

| 企業 | 分野 | 位置づけ |
| --- | --- | --- |
| レーザーテック | EUVマスク欠陥検査（アクチニック方式） | ほぼ100% |
| 扶桑化学 | CMP砥粒（コロイダルシリカ） | 世界90%超 |
| 信越化学 | シリコンウェハー | 世界最大手 |
| JX金属 | スパッタリングターゲット | 約60% |
| 三井化学 | EUVペリクル | 量産の中心企業 |
| 住友ベークライト | 封止材（EMC） | 世界シェア1位 |
| 三井ハイテック | リードフレーム | 世界シェア1位 |
| 味の素 | ABF（FC-BGA基板の層間絶縁材料） | ほぼ独占供給 |

一方で、バランスを取るために弱点も明示する。CMP用パッドは米DuPontが世界の約80%を占め、
日本勢は劣勢に立つ（詳細は `frontend` ページ）。また、荏原（CMP装置）や KOKUSAI ELECTRIC
（縦型炉）は概況扱いの数値・位置づけであり、確度高の断定枠には含めない。数値全体の確度
区分の考え方は `sources` ページに一覧化している。
