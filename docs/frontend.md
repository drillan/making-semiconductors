# 前工程 詳細

このページは、スライド本編「前工程（第2部）」を補足する。前工程はシリコンウェハーに
回路パターンを作り込む、半導体製造の中心工程であり、本編で最も多くの装置・材料企業が
登場する部でもある。各工程の目的を解説したうえで、付録Aの一次リストに基づく企業表を
工程ごとに整理し、マスク製造エコシステムと微細化・配線革新にも触れる。

## 前工程全体フロー

前工程は、成膜・リソグラフィ（露光）・エッチング・{term}`イオン注入`といった工程を、必要な回路層の
数だけ繰り返す。1枚のウェハーが数百の工程を経ることも珍しくない。

```{mermaid}
:caption: 前工程の繰り返しフロー
:align: center

flowchart LR
  A["成膜"] --> B["リソグラフィ"] --> C["エッチング"] --> D["イオン注入"] --> E["CMP平坦化"] --> F["洗浄"] --> G["検査・計測"] --> A
```

各工程の合間には必ず洗浄・計測が挟まり、微小なパーティクルや欠陥が次の層に持ち越されない
よう管理する。工程数が多いほど歩留まり管理の重要性は増す。

## ①ウェハー製造

シリコンインゴットを薄くスライスし、鏡面研磨したシリコンウェハーが、全工程の出発点となる
基板。単結晶シリコンの純度・平坦度が、その後すべての工程の品質を左右する。

| 企業 | 役割 | 位置づけ |
| --- | --- | --- |
| 信越化学 | シリコンウェハー製造 | 世界最大手（確度高） |
| SUMCO | シリコンウェハー製造 | 信越化学と並ぶ大手（確度高） |

## ②成膜（CVD／PVD／ALD）と装置

ウェハー上に絶縁膜・導電膜を積み重ねる工程。化学反応で膜を形成する{term}`CVD`（Chemical Vapor
Deposition）、物理的にターゲット材を飛ばして堆積させる{term}`PVD`（Physical Vapor Deposition）、
原子層単位で極めて薄い膜を精密制御する{term}`ALD`（Atomic Layer Deposition）を、必要な膜質・
微細度に応じて使い分ける。

| 企業 | 手法 | 位置づけ |
| --- | --- | --- |
| AMAT | CVD/PVD 等 | 成膜装置全般で大手 |
| TEL（東京エレクトロン） | CVD 等 | 成膜装置で大手 |
| Lam Research | CVD 等 | 成膜装置で大手 |
| ASM International | ALD | ALD装置に強み |

微細化が進むほど膜厚制御の精密さが要求され、原子層単位で成膜できるALDの重要性が増している
（後述のモリブデンALD配線もこの延長線上にある）。

## 熱処理・アニール

酸化・拡散・{term}`RTP`（Rapid Thermal Processing、急速熱処理）など、ウェハーに熱を加えて膜質や
不純物分布を制御する工程。バッチ処理に向く縦型炉が広く使われる。

| 企業 | 装置 | 位置づけ |
| --- | --- | --- |
| TEL / KOKUSAI ELECTRIC / ASM International | 縦型炉 | 3社で世界シェア73%超 |

```{admonition} 企業別シェアの序列は断定しない
:class: note

熱処理・縦型炉は TEL・KOKUSAI ELECTRIC・ASM International の3社で世界シェア73%超という
寡占構造までは複数の調査で裏付けが取れている。しかし**この3社の中でどの企業が首位かという
企業別の序列**は、調査時点で出典間の一致が得られておらず、本編・本ドキュメントとも断定しない
（{doc}`sources` ページ「スコープ外」参照）。ALD装置についても同様に、カテゴリ全体の重要性は
扱うが企業別シェア序列は断定しない方針を取る。
```

## 成膜の材料

成膜には高純度の前駆体・特殊ガスに加え、PVDで使う{term}`スパッタリング`用のターゲット材が使われる。

| 分野 | 代表企業 | 位置づけ |
| --- | --- | --- |
| 前駆体・特殊ガス | 関東電化 / 大陽日酸 / レゾナック / Air Liquide / Merck | 高純度ガス・薬液を供給する専業各社（概況） |
| スパッタリングターゲット | JX金属 | 世界シェア約60%（確度高） |

前駆体・特殊ガスの分野は特定1社が突出するのではなく、日系・欧米系の専業各社が用途ごとに
棲み分ける構造になっている。一方、スパッタリングターゲットはJX金属が高いシェアを持つ
「日本の強み」の一例。

## ③リソグラフィ／露光

パターンを転写する工程。感光材（レジスト）を塗布したウェハーにArF液浸光やEUV光でパターン
を焼き付け（露光）、現像する。

```{mermaid}
:caption: リソグラフィの流れ
:align: center

flowchart LR
  R["レジスト塗布"] --> E["露光"] --> D["現像"]
```

| 企業 | 役割 | 位置づけ |
| --- | --- | --- |
| ASML | {term}`EUV`/DUV露光装置 | EUVは唯一の供給元（代替なし）。High-NA機も展開（確度高） |
| Nikon / Canon | DUV露光装置 | 成熟ノードで競合。ArF液浸/KrF等、各社シェアは概況で約5%程度。EUVは非供給（確度高：EUV非供給の事実／概況：個別シェア%） |
| TEL | コータ・デベロッパ | EUV用装置でほぼ100%（確度高） |
| JSR / 東京応化 / 信越化学 | {term}`フォトレジスト` | 日本勢が世界シェアの大半（8〜9割程度）を占める（概況） |

最先端ロジックのEUV露光はASMLが世界で唯一の供給元であり、代替供給元が存在しない点は複数の
一次情報で一致している。一方、成熟ノード（レガシーノード）向けのDUV露光装置では Nikon・
Canon が競合として存在感を持つが、両社ともEUV機は供給していない。

## マスク製造エコシステムと日本の強み

フォトマスク（レチクル）は、露光のたびに回路パターンを転写する「原版」。マスクの描画・
検査・素材（ブランクス・ペリクル）のいずれも日本企業の存在感が大きく、本編全体を通じても
「日本の強み」が最も色濃く表れる領域。

| 企業 | 役割 | 位置づけ |
| --- | --- | --- |
| NuFlare | マスク描画装置（電子ビーム） | マルチビーム方式で独占的（確度高） |
| レーザーテック | EUVマスク欠陥検査 | アクチニック（EUV光源）方式でほぼ100%。e-beam方式の検査装置は他社も存在（確度高） |
| AGC | {term}`マスクブランクス` | 世界シェア約59%（概況） |
| HOYA | マスクブランクス | 世界シェア約34%（概況、AGCと2社合計で約93%） |
| 三井化学 | {term}`EUVペリクル` | 量産を担う中心企業（確度高） |

マスク描画装置（電子ビーム露光機）で原版にパターンを描き、マスクブランクス（石英基板に
遮光膜を成膜した素材）に転写し、完成したマスクは微小異物からの汚染を防ぐ薄膜「ペリクル」
で保護する。この一連のエコシステムの各段階で日本企業が中心的な役割を担っている点が特徴的。
特にレーザーテックのアクチニック（EUV光源を用いる）方式のマスク欠陥検査は、EUVマスクの
量産に不可欠な検査能力として、ほぼ独占的な地位を確立している。

成膜・露光で形成したレジストパターン通りに、不要な膜を化学的・物理的に削り取る工程。
反応性ガスのプラズマを使う{term}`ドライエッチング`が、微細なパターンの精度を確保する主流手法。

```{mermaid}

:align: center

flowchart LR

```

| 企業 | 役割 | 位置づけ |
| --- | --- | --- |
| Lam Research | ドライエッチング装置 | エッチング装置で大手 |
| TEL | ドライエッチング装置 | エッチング装置で大手 |

## ⑤イオン注入・ドーピング

シリコン結晶に不純物イオンを高速で打ち込み、電気的性質（n型／p型）を作り込む工程。加速した
イオンビームを狙った深さ・濃度でウェハーに注入する。

```{mermaid}
:caption: イオン注入の流れ
:align: center

flowchart LR
  I["イオン源"] --> A["加速・質量分離"] --> W["ウェハーへ注入"]
```

| 企業 | 役割 | 位置づけ |
| --- | --- | --- |
| AMAT | イオン注入装置 | 世界首位（確度高。ただし具体的なシェア%は再確認対象） |
| Axcelis | イオン注入装置 | 専業大手 |
| 住友重機械（SEN） | イオン注入装置 | 主要プレイヤーの一角 |

```{admonition} 再確認対象：AMATのイオン注入シェア%
:class: warning

AMATがイオン注入装置で世界首位であること自体は確度高だが、シェアの具体的な数値（調査時点
の一部資料にある「約38%」）は一般的な報告（過半〜7割とする資料が多い）と乖離があり、
出典の確定ができていない。そのため本編・本ドキュメントとも、AMATのイオン注入シェアは
**数値を明記せず「世界首位」という定性的な表現にとどめている**。詳細は {doc}`sources` ページ
「再確認対象」参照。
```

## ⑥CMP 平坦化

成膜を重ねるほど生じる表面の凹凸を、化学反応とスラリー研磨で平坦にする工程。次の層を精度
良く形成するために不可欠な工程で、{term}`CMP`装置・スラリー・砥粒・パッドという材料階層が存在する。

```{mermaid}
:caption: CMPの流れ
:align: center

flowchart LR
  S["凹凸のある表面"] --> C["化学反応+研磨"] --> F["平坦な表面"]
```

| 分野 | 代表企業 | 位置づけ |
| --- | --- | --- |
| CMP装置 | AMAT / 荏原 | 荏原は世界2位級（概況） |
| スラリー完成品 | Entegris / DuPont / フジミ | 研磨液を仕上げる完成品メーカー各社（概況） |
| CMP砥粒（{term}`コロイダルシリカ`） | 扶桑化学 | 世界シェア90%超（確度高、複数ソース一致） |
| CMPパッド | DuPont | 世界シェア約80%（概況。日本勢は手薄で、本編で明示する数少ない弱点の一つ） |

CMP工程は、装置（研磨機本体）・スラリー（研磨液の完成品）・砥粒（スラリーに含まれる研磨材
の原料）・パッド（ウェハーを研磨する当て材）という4つの階層に分けて理解すると、日本企業の
強み・弱みの非対称性が見えやすい。砥粒（扶桑化学）では世界的な強みを持つ一方、パッドでは
米DuPontが約80%を握り、日本勢は劣勢に立つ。同じCMP工程の中でも階層によって競争力が異なる
点は、「日本の強み」を一面的に語らないための重要な対比材料。

各工程の合間にウェハー表面のパーティクルや金属汚染を除去する工程。ウェハーを1枚ずつ処理

## ⑧計測・検査（メトロロジー）

各工程の後にパターン寸法・欠陥・膜厚などを測定し、歩留まりを支える工程。微細化が進むほど
計測精度への要求が上がり、業界内での重要性が増している分野。

| 企業 | 役割 | 位置づけ |
| --- | --- | --- |
| KLA | 計測・検査装置全般 | プロセス制御分野の首位 |
| 日立ハイテク | {term}`CD-SEM`（寸法計測） | 寸法計測で主要プレイヤー |
| Onto Innovation / Nova / Bruker | 計測装置各種 | 専業各社 |

計測・検査市場は、装置投資全体（{term}`WFE`: Wafer Fab Equipment）の約14%を占めるとされる
（2025年頃の概況）。米NIST（国立標準技術研究所）は{term}`CHIPS法`のもとで、微細化に計測能力が
追いついていない「計測ギャップ」を国家的な課題として位置づけている。

```{admonition} 再確認対象：KLAの計測・検査指標%
:class: warning

KLAは計測・検査（プロセス制御）分野の首位であること自体は確度高だが、「17%超」といった
一部資料の指標は、市場全体に占めるWFE比率（約14%）とKLA単体のプロセス制御シェア（5割超
とされる）が混同されている疑いがあり、どちらを指す数字か出典を確定できていない。そのため
本編・本ドキュメントとも、KLAの具体的な指標%は明記していない。詳細は {doc}`sources` ページ
「再確認対象」参照。
```

## 微細化と配線革新

トランジスタの微細化は、平面構造から{term}`FinFET`、さらにチャネル全周をゲートが囲む{term}`GAA`
（Gate-All-Around、ナノシート構造）へと進化している。配線側でも新たな技術が焦点になって
いる。

| トピック | 内容 |
| --- | --- |
| トランジスタ構造 | FinFET → GAA（ナノシート）へ移行 |
| バックサイド給電（{term}`BSPDN`） | 電源配線をウェハー裏面に移し、表面側の配線混雑を緩和する |
| {term}`High-NA` EUV | ASMLの次世代露光技術。さらなる微細化を可能にする |
| 配線材料の刷新 | 微細配線でW（タングステン）からモリブデンALDへの移行が進む（Lam ALTUS Halo, 2025年） |

微細化が進むほど配線の断面積が縮小し、従来主流だったタングステン配線では電気抵抗の増大が
性能のボトルネックになる。モリブデンをALD（原子層堆積）で成膜する新手法は、タングステンに
比べてバリア層を薄く（あるいは不要に）できるため低抵抗化に寄与するとされ、Lam Research が
2025年に発表したALTUS Haloはこの流れを象徴する装置である。ただし、この分野の企業別シェア
序列については前述の熱処理・ALD装置と同様に断定しない方針を取る。
