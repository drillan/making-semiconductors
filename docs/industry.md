# 業界・関連企業

このページは、スライド本編「業界俯瞰（第4部）」を補足する。ここまで見てきた設計・前工程・
後工程を1枚の地図に統合したうえで、装置・素材業界の全体構造、ファウンドリ・メモリ・
パワー半導体という業界横断のテーマ、そして地政学的な論点を扱う。

## 製造装置業界のシェア構造

半導体製造装置市場は少数の大手に集中している。上位5社の顔ぶれと序列はほぼ動かない一方、
装置カテゴリごとに強い企業は異なる。

| 企業 | 主な装置カテゴリ | 位置づけ |
| --- | --- | --- |
| ASML | EUV/DUV露光 | 装置売上トップ、EUVは事実上独占（確度高） |
| AMAT | 成膜・イオン注入 等 | 幅広いカテゴリを持つ総合装置最大手級 |
| Lam Research | エッチング 等 | 世界2位級の一角 |
| 東京エレクトロン（TEL） | コータ・デベロッパ 等 | 日本最大手、世界4位級 |
| KLA | 計測・検査 | プロセス制御で首位級 |

装置売上トップ5の序列（ASML > AMAT > Lam > TEL > KLA）は、複数の一次情報・調査が一致
しており確度高で断定できる。上位5社でトップ10社の売上合計の約85%を占めるとされ（2024年頃
の概況）、装置業界の寡占度は非常に高い。カテゴリ別の詳細（成膜・露光・エッチング・CMP・
洗浄・計測など各工程の主要装置企業）は {doc}`frontend` ページを参照。

## 素材業界と日本の位置づけ

半導体材料は装置ほど目立たないが、世界市場の約48%を日本企業が占めるとされ（概況推計）、
前工程・後工程を通じて日本の存在感が際立つ分野になっている。

| 分野 | 代表企業 | 位置づけ |
| --- | --- | --- |
| フォトレジスト | JSR / 東京応化工業 / 信越化学 | 世界シェア8〜9割（概況） |
| シリコンウェハー | 信越化学 / SUMCO | 世界最大手級（確度高） |
| スパッタリングターゲット | JX金属 | 約60%（確度高） |
| マスクブランクス | AGC / HOYA | 2社で約93%（概況） |
| 封止材（EMC） | 住友ベークライト | 世界シェア1位（確度高） |

材料世界市場に占める日本企業の約48%というシェアは民間調査の推計に基づく概況値であり、
個別企業のシェア%（AGC・HOYAの内訳など）は概況として丸めて記載している。フォトレジスト・
シリコンウェハー・封止材のように複数ソースが一致する確度高の項目と、内訳%が出典間で
幅を持つ概況の項目が混在している点に留意されたい（詳細は {doc}`sources` ページ参照）。

## 日本の強み総まとめ

これまでの各工程で見た日本企業の強みを、確度の高いものからまとめる。

**確度高で断定できる「隠れたチャンピオン」**

| 企業 | 分野 | 位置づけ |
| --- | --- | --- |
| レーザーテック | EUVマスク欠陥検査（アクチニック方式） | ほぼ100% |
| 扶桑化学 | CMP砥粒（コロイダルシリカ） | 世界90%超 |
| 信越化学 | シリコンウェハー | 世界最大手 |
| JX金属 | スパッタリングターゲット | 約60% |
| 三井化学 | EUVペリクル | 量産で強み |
| 三井ハイテック / 住友ベークライト | リードフレーム / 封止材 | 各分野で世界シェア1位 |
| 味の素 | ABF（FC-BGA層間絶縁材料） | ほぼ独占供給 |

**概況（参考情報、断定は避ける）**

| 企業 | 分野 | 位置づけ |
| --- | --- | --- |
| 荏原 | CMP装置 | 世界2位級（概況） |
| KOKUSAI ELECTRIC | 縦型炉（熱処理） | 3社寡占の一角（概況、単独首位ではない） |

一方で弱点もある。CMP用パッドは米DuPontが世界の約80%を占め、日本勢（富士紡等）は劣勢に
立つ。この非対称性——完成品の最終製造では存在感が薄い一方、特定の素材・専業装置・マスク
周辺では隠れたチャンピオンが多く存在する——が、日本の半導体産業における立ち位置を理解する
うえでの軸になる。

```{note}
確度高の項目は複数の一次情報・調査が一致する数値。概況（荏原・KOKUSAI ELECTRIC）とCMP
パッドの数値は2024年頃の民間調査推計であり、実装時点での再確認を経ても出典間の幅が残る
ため「概況」の扱いを維持している。
```

## ファウンドリ勢力図

先端ロジックの製造受託（ファウンドリ）は、TSMCが世界シェアの過半を占め、Samsung・Intelが
追う構図。最先端の2〜3nmノードを巡る競争が業界の焦点になっている。

| 企業 | シェア・位置づけ | 先端ノード |
| --- | --- | --- |
| TSMC | 世界シェア過半 | 2〜3nmで先行 |
| Samsung | 大手2番手級 | 2〜3nmを量産 |
| Intel | IDM兼ファウンドリ展開 | 先端ノードを開発中 |
| Rapidus | 日本、2nm量産を目標 | パイロットラインを整備中 |

ファウンドリシェアの序列（TSMCが過半、Samsung・Intelが追う構図）は複数ソースが一致する
確度高の情報。

```{admonition} Rapidus×Tenstorrentの関係：IP共同開発提携であり製造委託ではない
:class: important

Rapidusは日本発の新興ファウンドリで、2nmロジックの量産を目標に掲げている。ここで注意すべき
なのは、RapidusとTenstorrent（RISC-Vベースのプロセッサを手がける半導体企業）の関係である。
両社は**2023年11月に2nmロジックのIP共同開発提携**を締結したが、これは**Tenstorrentが
Rapidusに製造を委託する契約ではない**。あくまで2nmプロセス向けのIP（回路設計資産）を
共同で開発する提携であり、Rapidusの2nmファウンドリ事業としての量産実績・顧客獲得とは
別の話である。この点は報道等で誤解されやすいため、本編・本ドキュメントとも正確な区別を
明記している。RISC-Vエコシステムにおける両社の位置づけは {doc}`design` ページ「EDA・IP・
ISA」も参照。
```

## メモリ

メモリはDRAM・NAND・HBMの3市場で構図が異なる。DRAMとNANDは大手数社の寡占、生成AI需要で
急拡大したHBMはSK hynixが首位を走る。

| 分野 | 主要企業 | 位置づけ |
| --- | --- | --- |
| DRAM | Samsung / SK hynix / Micron | 大手3社の寡占 |
| NAND | Samsung / Kioxia / SK hynix / Micron 等 | 層数競争が焦点 |
| HBM（広帯域メモリ） | SK hynix（首位）/ Samsung / Micron | AI需要を背景に急成長 |

HBMは、DRAMを複数枚積層してAI半導体向けに帯域を拡大したメモリ製品で、その積層構造の
詳細は {doc}`backend` ページ「HBM・積層化・3D実装」で扱う。NANDの「層数競争」は、平面的な
微細化が限界に近づく中、垂直方向に記憶セルを積み重ねることで記録密度を高める技術トレンド
を指す。

## パワー半導体の別サプライチェーン

自動車・産業機器向けのパワー半導体は、先端ロジックとは別の設計・供給網を持つ。本編の主対象
である先端ロジック・メモリとは異なるサプライチェーンとして、業界俯瞰の文脈でのみ扱う
（製造フローの詳細な解説はスコープ外）。

| 分野 | 代表企業 | 位置づけ |
| --- | --- | --- |
| パワー半導体（総合） | Infineon / STMicro / onsemi / ローム / 三菱電機 | 車載・産業機器向けの大手 |
| SiC基板 | Wolfspeed / Coherent / ローム | 次世代SiCデバイスを支える基盤材料 |

パワー半導体は微細化競争よりも、耐圧・耐熱・省電力といった特性が重視される分野で、
先端ロジックのようなEUV露光やチップレット技術は基本的に登場しない。次世代材料である
SiC（炭化ケイ素）・GaN（窒化ガリウム）などの化合物半導体基板の供給網は、シリコン中心の
先端ロジックとは異なる顔ぶれの企業が並ぶ。

## 地政学・供給リスク

半導体は経済安全保障の中心的な論点になっている。先端製造が特定地域に集中し、米中の技術
覇権争いを背景に各国が自国内生産の誘致を競っている。

| 論点 | 概要 | 影響 |
| --- | --- | --- |
| 台湾集中 | 先端ロジック製造の大半をTSMCなど台湾勢が担う | 供給網の地政学リスクの焦点 |
| 米中対立 | 先端装置・技術の輸出規制が強化 | サプライチェーン分断・多極化の懸念 |
| 各国補助金 | 米CHIPS法など、自国内製造誘致策 | 日米欧・Rapidus含め国内投資が加速 |

台湾への先端製造の集中は、自然災害・地域紛争などのリスクが顕在化した場合、世界のAI・
先端ロジック供給に与える影響が極めて大きいことを意味する。米CHIPS法をはじめ、各国が
自国・地域内での製造能力確保に補助金を投じる動きが強まっており、日本のRapidusもこの
文脈の中に位置づけられる（Rapidusの詳細は前述「ファウンドリ勢力図」参照）。

## トレンド

生成AIの普及が半導体業界の技術トレンドを牽引している。演算性能の向上には、チップ単体の
微細化だけでなく、メモリ帯域や実装技術の進化が欠かせない。

| トレンド | 概要 | 位置づけ |
| --- | --- | --- |
| AI半導体 | 生成AI向け需要が急拡大 | NVIDIA等の成長を牽引 |
| HBM | 広帯域メモリの需要急増 | SK hynix等が先行 |
| 先端パッケージング | CoWoS等、2.5D/3D実装の普及 | 性能向上のカギ |
| 積層化 | TSV・ハイブリッドボンディング | 微細化の限界を補う技術 |

これらのトレンドは独立した現象ではなく、相互に連動している。生成AI向け需要の急拡大が
HBMの需要を押し上げ、HBMをロジックダイと統合するために先端パッケージング（CoWoS等）の
重要性が増し、その先端パッケージングを実現する基盤技術として積層化（TSV・ハイブリッド
ボンディング）が発展する、という因果関係で捉えると理解しやすい。

## まとめ

設計から前工程・後工程、そして業界俯瞰まで、半導体産業は工程ごとに専業化した企業群が
国際分業でつながる構造を持つ。装置・素材・後工程材料の随所に、確度高で断定できる日本の
「隠れたチャンピオン」が存在する一方、CMPパッドのような弱点も残る。

| 部 | 主なポイント | 位置づけ |
| --- | --- | --- |
| 設計 | ファブレス・EDA・IP・オープンISA | 分業構造の起点 |
| 前工程 | 装置・材料に日本企業が多数存在 | 微細化競争の中心 |
| 後工程 | 先端パッケージング拡大、材料に強み | 性能向上のカギを担う |
| 業界俯瞰 | ファウンドリ・メモリ・パワー・地政学 | 日本の強みと弱みが併存 |

数値・シェアの根拠と確度区分の詳細な一覧は {doc}`sources` ページに、専門用語の解説は
{doc}`glossary` ページに集約している。
