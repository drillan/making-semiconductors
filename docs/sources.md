# データ出典

このページは、スライド本編（`slides/`）および {doc}`overview`・{doc}`design`・{doc}`frontend`・
{doc}`backend`・{doc}`industry` の各詳細ページで扱う数値・企業シェアの出典と確度区分を
一箇所に集約する。各ページの出典行・注記から「詳細は `sources` ページ参照」と示された項目は、
すべてこのページのいずれかの節に対応する。

```{note}
シェア・市場データは変動する。本ページで示す数値はすべて特定時点（取得年）の値であり、
将来にわたって固定された値ではない。参照する際は必ず取得年とあわせて解釈すること。
特に「概況」区分の数値は民間調査の推計であり、出典間で幅がある点に留意する。
```

## 確度区分の考え方

本編・本ドキュメントは、すべての数値・企業の位置づけを次の3段階のいずれかに区分して扱う。

確度高
: 複数の一次情報・独立した調査が一致して裏付ける数値・位置づけ。断定的に記述する。

概況
: 民間調査の推計や、出典間で幅がある数値。「約」を付けて丸め、断定を避ける。

再確認対象
: 実装時点で出典を確定できなかった数値。数値そのものは明記せず、定性的な位置づけのみを
  記述する（下記「再確認対象」節を参照）。

## 再確認対象（数値未確定・数値を断定しない）

以下の4項目は、実装時点で一次情報・複数調査の一致が取れず、**数値そのものを明記していない**。
定性的な位置づけ（首位である、寡占の一角である、等）のみを本編・本ドキュメントで扱う。

AMATのイオン注入シェア%
: AMATがイオン注入装置で世界首位であること自体は確度高。しかし調査時点の一部資料にある
  「約38%」という数値は、一般的な報告（過半〜7割とする資料が多い）と乖離があり、出典を
  確定できていない。詳細は {doc}`frontend`「⑤イオン注入・ドーピング」参照。

KLAの計測・検査指標%
: KLAが計測・検査（プロセス制御）分野の首位であること自体は確度高。しかし「17%超」等の
  一部資料の指標は、市場全体に占めるWFE比率（約14%）とKLA単体のプロセス制御シェア
  （5割超とされる）が混同されている疑いがあり、どちらを指す数字か確定できていない。
  詳細は {doc}`frontend`「⑧計測・検査（メトロロジー）」参照。

住友ベークライトによる京セラEMC事業買収（2026年）
: 住友ベークライトが世界シェア1位であること自体は既存の複数ソースで裏付けられている。
  一方、2026年に報じられた京セラのEMC事業買収については、実装時点で新しい情報であり、
  買収範囲・完了時期・市場シェアへの影響について最終確認ができていない。詳細は
  {doc}`backend`「ボンディング・封止＋装置」参照。

Heraeusのボンディングワイヤ全体シェア%
: 日本の田中電子工業（金線）・日鉄マイクロメタル（銅線）が個社としては主要プレイヤーだが、
  金線・銅線を含めた市場全体ではドイツのHeraeusが首位という位置づけ自体は複数の資料で
  確認できる。ただし、その具体的なシェア%は出典間で確定できていない。「Heraeusが全体首位」
  という位置づけまでを確度の範囲として扱う。詳細は {doc}`backend`「ボンディング・封止＋装置」参照。
