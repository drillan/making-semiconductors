## 1. 目的とゴール

半導体の設計から製造（前工程・後工程）、そして装置・素材までのバリューチェーンを
**体系的に理解できる**スライドを作る。加えて、各工程に関わる**実企業と市場シェア**を
織り込み、**業界俯瞰（ビジネス視点）**も同時に得られるようにする。

### 想定読者・用途
- A. 社内・チーム向けの勉強会 / 入門教材（業界を知らない人が全体像を掴む）
- B. 投資・ビジネス視点の業界俯瞰（企業・サプライチェーン・シェア重視）

「なぜ半導体が重要か」は既知の前提として省き、代わりに半導体の**種類**紹介から入る。

### 成功基準
- 半導体製造のバリューチェーンを、順を追って一通り説明できる
- 各工程で「誰が（どの企業が）主要プレイヤーか」が分かる
- 装置・素材・ファウンドリ・メモリの勢力図を数字付きで俯瞰できる
- **日本企業が強い領域（材料・マスク周辺・特定装置）を軸に、業界の構造が語れる**
- すべての市場データに取得年と出典が明記されている

## 2. 全体方針

- 構成の主軸: **バリューチェーン順（工程フロー主軸）**。企業・シェアは各工程内に織り込み、
  終盤の「業界俯瞰」節で総まとめする。
- ボリューム: **現状 約45枚**（タイトル＋本文44）（当初34枚から、網羅性調査を反映して拡張）。**上限は設けない** —
  内容の充実・工程やプレイヤーの追加に応じて増やしてよい。勉強会は 75〜90分を目安（枚数増に応じ延長可）。
- 制作ツール: **sphinx-revealjs**（MyST Markdown → HTML スライド）。スライド本体は `slides/`、
  詳細・出典・用語集の補助ドキュメントは `docs/` に分離する2プロジェクト構成（§6）。
- 本編の主対象は**先端ロジック / メモリ**。パワー半導体は種類紹介で触れ、業界俯瞰で別系統の
  サプライチェーンとして1枚扱う（EUV・チップレット等の本編フローは先端ロジック/メモリの話）。
- **「日本の強み」を通底テーマ**にする（材料で世界約48%、マスクブランクス/ペリクル、フッ素系
  ガス、スパッタターゲット、コロイダルシリカ砥粒、封止材など）。弱点（CMPパッド＝米DuPont等）
  も対比で示し、一面的な「日本礼賛」に寄らないようバランスを取る。

## 4. 表現方法・スライドの約束事

- **1スライド1メッセージ**。工程スライドは定型3ブロック:
  ①何をする工程か（1〜2行）→ ②図 → ③主要企業＋シェア
- **図解は Mermaid**（sphinx-oceanid 経由）。フロー図中心。鳥瞰図・前工程フロー・後工程フローで多用
- **企業・シェアは list-table（最大3列厳守）**。「企業 / 役割 / シェアや位置づけ」の形。
  4項目以上を比べたい場合は属性を統合するか2枚に分割する（スライド2がこの典型）
- **出典表記はMySTの脚注機能を使わない**。MyST脚注はセクション末尾にまとめて描画され、
  「各スライド下部に出典」にならないため。代わりに各スライド末尾に**小文字の出典テキスト行**を置き、
  カスタムCSSクラス（例 `.source`）で小さく表示する。数値の隣に取得年も併記する。
  **完全な出典URL一覧は docs の `sources` ページに集約**し、スライドの出典行は簡潔な要約＋docsへの導線に留める
- **色の約束**: 設計＝青、前工程＝緑、後工程＝橙、素材＝紫。鳥瞰図と個別スライドを視覚的に接続
- **日本企業はバッジ/マーカーで強調**（「日本の強み」テーマを視覚的に一貫させる）
- 上記の色分け・バッジ・出典行はいずれも**カスタムSCSSクラス前提**（§6参照）

## 5. データの正確性・調査方針

- シェア・市場規模・企業の位置づけは **Web 調査で取得**済み（付録A・根拠調査レポートが基）。
  実装時に最終数値を再確認する。
- **すべての数値に「取得年＋出典」を明記**。最終スライドの「データ出典」節に集約
- 不確実・幅のある数字は「約○%」「〜年時点」「概況」と丸め、断定を避ける
- 確度の区別（実装時に踏襲）:
  - **確度高（一次情報・複数ソース一致で断定可）**: 装置トップ5順位（ASML>AMAT>Lam>TEL>KLA）、
    レーザーテック＝アクチニックEUVマスク検査ほぼ独占、TEL＝コータ/デベロッパほぼ独占、
    扶桑化学＝コロイダルシリカ砥粒90%超、JX金属＝スパッタターゲット約60%、
    三井化学＝EUVペリクル量産、住友ベークライト＝封止材世界1位、三井ハイテック＝リードフレーム世界1位、
    信越化学＝シリコンウェハー世界最大、
    ABF＝味の素のパッケージ基板絶縁材ほぼ独占、Nikon/Canon＝DUV競合でEUV非供給
  - **概況（民間調査推計・出典間で幅、%は丸める）**: CMPスラリー/パッドの個社%、EMC日本合算%、
    ボンディングワイヤ合算%、前駆体個社%、EUVブランクスのAGC/HOYA内訳%、Nikon/Canonの個別シェア%
- **実装前に必ず再確認する数値（疑義あり）**:
  - イオン注入の **AMATシェア「約38%」は低すぎる疑い**（一般に過半〜7割との報告が多い）。要出典確定
  - 計測・検査の **KLA「17%超」は指標が不明瞭**（市場のWFE比率≈14% と KLAのプロセス制御シェア
    ≈5割超 が混在の疑い）。どちらの数字か出典に戻って確定
  - 熱処理は **KOKUSAI単独ではなく「縦型炉 TEL/KOKUSAI/ASM International 3社で73%超」**が裏付け範囲
  - 住友ベークライトによる **京セラEMC事業買収（2026年発表）** は新情報のため最終確認
- **要注意の訂正点**: ナミックスはレゾナック傘下ではなく独立系（アンダーフィルで競合）。
  レゾナックが統合したのは日立化成。

## 6. 技術構成（2プロジェクト構成）

スライド本体と補助ドキュメントを**別々のSphinxプロジェクト**に分ける。

### 6.1 `slides/` — プレゼン本体（sphinx-revealjs）
- `slides/` は**手動スキャフォールド**する（`revealjs-init` は出力先が `docs/` 固定で、`docs/` を
  補助ドキュメントに充てる本構成と衝突するため使わない）。MyST Markdown
- **単一デッキ構成**：sphinx-revealjs はソース文書1つにつきプレゼンHTMLを1つ生成するため、
  `slides/index.md` に各部を `{include}` で統合し、部ファイル（`00-intro.md`〜）は `exclude_patterns`
  で単独ビルドを抑止する。`#`=タイトルスライドは index.md 専用、部ファイルは `##` 始まり
- **テーマ = solarized**（`revealjs_style_theme = "solarized"`、reveal.js 標準）
- **SCSSカスタムテーマオプションを有効化**（§4の色分け・日本企業バッジ・出典行クラスに必須）。
  工程色は solarized アクセントに合わせる：設計=青#268bd2 / 前工程=緑#859900 /
  後工程=橙#cb4b16 / 素材=菫#6c71c4
- **Mermaid 有効化**（sphinx-oceanid）、シンタックスハイライトCSS（必須）
- 見出し階層でスライド構造: `#`=タイトルスライド / `##`=各部の区切り / `###`=個別スライド
- ビルドは `sphinx-build` スキル（`make revealjs` / ライブプレビュー `livehtml`）
- スライド原稿は部ごとにファイル分割（`slides/00-intro.md` 〜 `slides/04-industry.md`）

### 6.2 `docs/` — 補助ドキュメント（Sphinx HTML、Furo テーマ）
- 通常のSphinx HTMLサイト。スライドが深掘りを委譲する先。**テーマ = Furo**
- 内容（標準スコープ）:
  - 各部の**詳細ページ5枚**（`design` / `frontend` / `backend` / `industry` / `overview`）。
    スライドを散文・表で補足し、確度区分の根拠を丁寧に説明
  - **データ出典・確度一覧ページ**（`sources`）。§5の確度区分と付録A・調査レポートの出典URLを集約
  - **用語集ページ**（`glossary`）
- 既存の `docs/superpowers/**`（spec/plan）は docs ビルドの `exclude_patterns` で除外する
- **相互リンク**: 各スライド末尾から docs の該当詳細ページ／出典ページへリンクし、
  スライド側の出典行は簡潔に保つ（詳細な出典はdocs集約ページに置く）
- ビルドは `sphinx-build` スキル（`make html` / `livehtml`）

## 7. スコープ外 / 非目標

- パワー半導体・化合物半導体（SiC/GaN）の製造フロー詳細（種類紹介＋業界俯瞰1枚での言及にとどめる。
  詳細を扱うには一次情報〔業界団体統計・各社IR〕での追加調査が必要）
- 個別企業の財務分析・投資推奨
- 各工程の物理・化学の深い理論（入門レベルの説明にとどめる）
- TDK（受動部品・磁性材料メーカーで前工程素材の主要プレイヤーでないため、本編では扱わない）
- 熱処理・ALD装置の**企業別シェア序列の断定**（調査で棄却済み。カテゴリの重要性と寡占構造のみ扱う）
