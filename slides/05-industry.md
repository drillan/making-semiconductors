### 代替の効かない寡占ボトルネック

露光(ASML)だけでなく、その前後工程にも単独企業しか供給できないボトルネックが点在する\
どれも代替が効かず、供給が止まると製造全体が止まる

:::{list-table}
:header-rows: 1
:widths: auto

* - 工程 / 品目
  - 企業
  - 状況
* - EUV露光装置
  - ASML
  - 唯一の供給元・代替なし
* - EUV光源
  - Cymer（ASML子会社）
  - 実質独占
* - 塗布・現像（コータ/デベロッパ）
  - [東京エレクトロン]{.jp}
  - EUV用でほぼ100%
* - EUVマスク欠陥検査
  - [レーザーテック]{.jp}
  - アクチニック方式でほぼ100%
* - マスク描画（マルチビーム）
  - [NuFlare]{.jp}
  - 独占的
* - FC-BGA絶縁材 ABF
  - [味の素]{.jp}
  - ほぼ独占（確度高）
* - 精密ダイシング・グラインド
  - [ディスコ]{.jp}
  - 準独占（7〜8割）
:::

:::{note}
2社寡占も実質ボトルネック\

### 用語集（要点）① 基礎

本編の略語・用語を1行で（詳細は docs 側の用語集ページ）

トランジスタ
: 電気のスイッチ チップはこれを数十億個ならべた回路

微細化・ノード
: トランジスタを小さくする世代（例 3nm/2nm）\
  小さいほど速く・安く・省電力

歩留まり
: 作ったチップのうち良品の割合

DRAM
: 電源を切ると消える主記憶用メモリ（揮発性）

NAND
: 電源を切っても保つ大容量ストレージ用メモリ（不揮発性）

レジスト
: 光が当たると性質が変わる感光材料 露光でパターンを転写する

### 用語集（要点）② 設計・前工程

EDA
: チップ設計を自動化するソフトウェア（Synopsys/Cadence等）

IDM
: 設計から製造まで自社一貫の垂直統合型企業

CVD・PVD・ALD
: ウェハーに薄膜を積む成膜手法（化学反応/物理的/原子層単位）

EUV
: 波長13.5nmの極端紫外線露光\
  最先端ロジック向けで供給元はASMLのみ

GAA
: FinFETの次世代トランジスタ構造\
  チャネル全周をゲートで囲む

BSPDN
: 電源配線をウェハー裏面に移し、表面の配線密度を高める技術

### 用語集（要点）③ 後工程・その他

HBM
: DRAMを積層し帯域を拡大したAI半導体向けメモリ

OSAT
: ダイシング・封止・テスト等の後工程を専業で請け負う企業群

FC-BGA
: 先端ロジック向けの高密度パッケージ基板（フリップチップBGA）

ABF
: FC-BGAの層間絶縁材料 味の素がほぼ独占供給

UCIe
: チップレット間接続のオープン業界標準

RISC-V
: オープンソースの命令セット（ISA）ライセンスISAに対する選択肢
